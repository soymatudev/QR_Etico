Demostración de datos que el navegador puede mostrar

Proyecto: Hacking ético (presentación). Esta página no almacena ni envía tus datos. Pide permiso y muestra solo información no sensible.

¿Te gustaría ver qué información puede leer el navegador si das permiso? Se mostrará solo: coordenadas (lat/lon), idioma, navegador, plataforma y tamaño de pantalla.

Pulsa Mostrar info y permite la ubicación si quieres ver el resultado. Si no permites, se mostrará la información del navegador sin coordenadas.

Nota: el "modelo" exacto del teléfono no siempre está disponible. El userAgent puede dar pistas (marca/OS), pero no debe usarse para identificar personas.

Notas éticas y legales (lee antes de probar)

  • Pide permiso explícito. Si la persona no da permiso, no insistas.
  • No almacenes ni compartas coordenadas ni userAgents sin consentimiento informado.
  • Hostea esta página en HTTPS (ej. GitHub Pages) para que la API de geolocalización funcione.
  • Usa la demo solamente con fines educativos y con sujetos que sepan que participan en una demostración.

Para la presentación: muestra la pantalla de consentimiento y luego el resultado en vivo para que los asistentes vean qué datos se muestran con permiso.
